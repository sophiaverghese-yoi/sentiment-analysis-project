# Sentiment Analysis Project

## Problem
People post many comments online, and it's hard to manually understand sentiment.

## Solution
This project uses machine learning to classify comments as positive, negative, or neutral.

## Goal
To help users quickly understand public opinion from text data.

---

## Features
- Sentiment classification (Positive, Negative, Neutral)
- Uses TF-IDF and Naive Bayes
- Includes accuracy and evaluation metrics

---

## Technologies Used
- Python
- Pandas
- Scikit-learn

---

## How to Run
1. Open the notebook in Google Colab
2. Upload `data.csv`
3. Run all cells

---

## Results
- Model performs basic sentiment classification
- Accuracy depends on dataset size
- Performance improves with more data
